== Description ==

The Media Folders plugin creates a "folder" custom post type, to which individual media items can be attached.  The Media Folder can then be tagged, categorized, and otherwise treated as a single entity.  All media items attached to the folder can be downloaded as a single zip file.

== Installation ==

1. Upload the 'media-folders' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
